# Customer-Ticket-Satisfaction
**Project Overview**
This project analyzes customer support ticket data to understand factors influencing customer satisfaction and builds a manchine learning model to predict satisfaction ratings.

This analysis includes data preprocessing, Exploratory data analysis(EDA),feature engineering, and model building using python.

# Objective
Analyze customer support ticket data
Identify factors affecting customer satisfaction
Build a machine learning model to predict satisfaction ratings.

# Tools & Technologies 
Python
Pandas
Numpy
Matplotlib
Seaborn
Scikit-Learn
Jupyter Notebook
Kaggle

# key Analysis
Includes :

1.Exploratory Data Analysis(EDA)
2.Correlation Heatmap
3.Ticket Distribution Analysis
4.Feature Engineering
5.Random Forest Classification Model
6.Model Evaluation using Confusion matrix and classification Report

# Manchine Learning Model
A Random Forest Classifier was used to predict customer satisfaction based on ticket and customer attributes.
